CSC 600: Programming Languages and Paradigms
Introduction to functional programming with TypeScript. Advanced topics include domain-specific languages, concurrency, and laziness. Exposure to interpreters and compilers.
CSC 825: Logic and Computation
Introduction to dependent type theory using the Coq Proof Assistant. Different computational models including finite state automata, Turing machines, and lambda calculus are covered.
