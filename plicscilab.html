Programming Languages in Computational Science Lab
The central question we study is: "what next-generation programming language abstractions and systems do we need to build to accelerate science research?" We tackle this question by building programming languages and systems with a focus on applications to computational chemistry and mathematics. Our work draws on techniques from programming languages, compilers, and machine learning.
Research
  • Programming Languages in Computational Chemistry. How can programming languages technology push the state-of-the-art in modeling atomistic systems (e.g., molecules, materials)? Computational chemistry is critical in the study of climate science (e.g., atmospheric chemistry and battery design for clean energy). Projects involve developing new languages, systems, algorithms, and machine learning models for accelerating atomistic modeling.
  • Programming Languages in Mathematics. How can programming languages technology push the state-of-the-art in modeling formal mathematics? Projects involve developing new languages, systems, and AI systems to do reasoning tasks: algebraic rewrites, euclidean geometry, construct proofs in a proof assistant.
